6.1 Complete las siguientes oraciones

  1. // comienza un comentario de una sola línea.
  2. Toda instucción de JavaScript debe teminar con un (a) punto y coma(;)
  3. La instuccón if se utiliza para tomar decisiones.
  4. El objeto window muestra diálogos alert y prompt.
  5. Las palabras clave están reservadas para uso de JavaScript.
  6. Los métodos write y writeln del objeto document escríben texto de HTML5 en un documento de HTML5.
